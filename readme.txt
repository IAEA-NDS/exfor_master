This repository contains the all EXFOR Master files restored from backup zip
files stored and maintained in the IAEA Nuclear Data Section. All data have 
been compiled by International Network of Nuclear Reaction Data Centres (NRDC).

See details in: https://nds.iaea.org/nrdc/
Data Retrieval System is avaialbe at: https://nds.iaea.org/exfor/

Please note that this is a repository for the internal test.
